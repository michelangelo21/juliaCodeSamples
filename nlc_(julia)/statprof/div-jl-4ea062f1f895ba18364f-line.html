39 """
40 rem(x, y, r::RoundingMode=RoundToZero)
41
42 Compute the remainder of `x` after integer division by `y`, with the quotient rounded
43 according to the rounding mode `r`. In other words, the quantity
44
45 x - y*round(x/y,r)
46
47 without any intermediate rounding.
48
49 - if `r == RoundNearest`, then the result is exact, and in the interval
50 ``[-|y|/2, |y|/2]``. See also [`RoundNearest`](@ref).
51
52 - if `r == RoundToZero` (default), then the result is exact, and in the interval
53 ``[0, |y|)`` if `x` is positive, or ``(-|y|, 0]`` otherwise. See also [`RoundToZero`](@ref).
54
55 - if `r == RoundDown`, then the result is in the interval ``[0, y)`` if `y` is positive, or
56 ``(y, 0]`` otherwise. The result may not be exact if `x` and `y` have different signs, and
57 `abs(x) < abs(y)`. See also [`RoundDown`](@ref).
58
59 - if `r == RoundUp`, then the result is in the interval `(-y,0]` if `y` is positive, or
60 `[0,-y)` otherwise. The result may not be exact if `x` and `y` have the same sign, and
61 `abs(x) < abs(y)`. See also [`RoundUp`](@ref).
62
63 """
