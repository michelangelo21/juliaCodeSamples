99 """
100 Threads.@threads [schedule] for ... end
101
102 A macro to parallelize a `for` loop to run with multiple threads. Splits the iteration
103 space among multiple tasks and runs those tasks on threads according to a scheduling
104 policy.
105 A barrier is placed at the end of the loop which waits for all tasks to finish
106 execution.
107
108 The `schedule` argument can be used to request a particular scheduling policy.
109 The only currently supported value is `:static`, which creates one task per thread
110 and divides the iterations equally among them. Specifying `:static` is an error
111 if used from inside another `@threads` loop or from a thread other than 1.
112
113 The default schedule (used when no `schedule` argument is present) is subject to change.
114
115 !!! compat "Julia 1.5"
116 The `schedule` argument is available as of Julia 1.5.
117 """
118 macro threads(args...)
119 na = length(args)
120 if na == 2
121 sched, ex = args
122 if sched isa QuoteNode
123 sched = sched.value
124 elseif sched isa Symbol
125 # for now only allow quoted symbols
126 sched = nothing
127 end
128 if sched !== :static
129 throw(ArgumentError("unsupported schedule argument in @threads"))
130 end
131 elseif na == 1
132 sched = :default
133 ex = args[1]
134 else
135 throw(ArgumentError("wrong number of arguments in @threads"))
136 end
137 if !(isa(ex, Expr) && ex.head === :for)
138 throw(ArgumentError("@threads requires a `for` loop expression"))
139 end
140 if !(ex.args[1] isa Expr && ex.args[1].head === :(=))
141 throw(ArgumentError("nested outer loops are not currently supported by @threads"))
142 end
143 return _threadsfor(ex.args[1], ex.args[2], sched)
144 end

146 """
147 Threads.@spawn expr
148
149 Create and run a [`Task`](@ref) on any available thread. To wait for the task to
150 finish, call [`wait`](@ref) on the result of this macro, or call [`fetch`](@ref)
151 to wait and then obtain its return value.
152
153 Values can be interpolated into `@spawn` via `\$`, which copies the value directly into the
154 constructed underlying closure. This allows you to insert the _value_ of a variable,
155 isolating the aysnchronous code from changes to the variable's value in the current task.
156
157 !!! note
158 See the manual chapter on threading for important caveats.
159
160 !!! compat "Julia 1.3"
161 This macro is available as of Julia 1.3.
162
163 !!! compat "Julia 1.4"
164 Interpolating values via `\$` is available as of Julia 1.4.
165 """
